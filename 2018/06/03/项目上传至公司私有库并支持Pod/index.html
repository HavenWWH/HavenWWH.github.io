项目上传至公司私有库并支持Pod

发表于 2018-06-03

1.创建私有库项目

pod lib create

详见创建私有库的文章

2.上传至gitHub 或者gitLab

详见git创建版本库命令

3.验证私有库文件

验证.podspec文件

pod lib lint CDHorizontalScrollView.podspec –allow-warnings

4.添加关联公司私有库(如果已关联则忽略此步骤)

$ pod repo add CDSpecs git@gitlab.ttsing.com:ios/CDSpecs.git

成功的话进入到本机~/.cocoapods/repos 查看是否有私有库

5.将个人私有库添加或者更新到公司私有库目录下

pod repo push CDSpecs CDHorizontalScrollView.podspec –allow-warnings

6.更新私有库(search不到的时候进行update)

pod repo update

7.验证是否支持Pod

pod search “name”

⚠️:

gitHub:

提交到CocoaPod 支持pod

pod trunk push CDHorizontalScrollView.podspec –allow-warnings

删除cocoapods的索引, 用于创建支持CocoaPod时 search不到的情况

rm ~/Library/Caches/CocoaPods/search_index.json
